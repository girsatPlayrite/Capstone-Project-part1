Data Acquisition, Cleaning, and Exploratory Analysis
For numeric columns with 20% or fewer missing values, missing entries were filled in using the column median. The median was preferred over the mean because it is less sensitive to outliers and skewed distributions — extreme values don't pull it in one direction the way they can with the mean — making it a more dependable measure of central tendency for real-world data. Columns with more than 20% missing values were flagged and handled separately, since such a high rate of missingness often calls for a different approach altogether, such as dropping the column, applying more advanced imputation methods, or conducting domain-specific analysis to determine the best course of action.
Descriptive Statistics and Skewness
Outlier Detection (IQR Method): Outliers were identified using the Interquartile Range (IQR) method. For each selected numeric column, the first quartile (Q1), third quartile (Q3), IQR, lower bound (Q1 − 1.5 × IQR), and upper bound (Q3 + 1.5 × IQR) were calculated. Observations outside these bounds were counted as outliers. No outliers were removed during this preprocessing stage to avoid discarding potentially meaningful observations.
